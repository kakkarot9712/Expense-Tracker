What is this?

This is Expense-Tracker app, User can add expenses with details and can keep track of.

    Features

  • This app shorts expenses according to year

  • This expense shows rough comparison of expenses using graphs

  • App will save expenses in local stroage and it will also retrieve expenses after reload.

  • User Can Modify or delete expenses



How Graphs are rendered?


-> Graphs won't give user the exact amount but it is pretty rough comparison.

-> Graphs are rough comparisons of expenses for every month for single year.

-> Graph will be made by taking highest expense amount as a base for all month's expense amount.

-> This is still useful as user don't have too look into every entry if what user wants to know is which month has highest expense and how expneses are spent on monthly basis.
